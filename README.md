# 🐾 Mascota Virtual

Este es un proyecto educativo desarrollado en **C++** para practicar conceptos básicos de programación como bucles, condicionales, manejo de variables y lógica de videojuegos por consola.

## 🚀 Funcionalidades
El programa simula una mascota virtual interactiva con las siguientes características:
- **Estadísticas en tiempo real:** Control de Hambre, Energía y Felicidad.
- **Sistema de Corazones:** La felicidad se visualiza mediante iconos de `<3`.
- **Sistema de Edad:** Tu mascota cumple años automáticamente cada 5 acciones realizadas.
- **Condiciones de Victoria/Derrota:** El juego termina si la mascota llega al límite de hambre o se queda sin energía.
- **Interfaz Limpia:** Uso de comandos para limpiar la pantalla y mejorar la experiencia del usuario.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** C++
- **Paradigma:** Programación Estructurada
- **Librerías estándar:** `iostream`, `string`, `cstdlib`.
